This program acts as a two player quantum connect four game through a single window. To place a piece press the space bar. To move a piece to the left or right press the respective arrow keys. To make a quantum move press Q, the same movement functions apply to quantum moves. Note, once half a quantum move has been placed one cannot switch and make a classical move. Additionally, quantum pieces can be placed up to two rows above the board if the given column is not full. After every five moves the quantum pieces will be measured, resulting in the fixing of their positions. 

There are three classes: Game, GameLogic and connectFourGraphics. The Game class combines the GameLogic and connectFourGraphics classes into a small class that forms the connect four game. The connectFourGraphics class handles all of the graphics related to the program, it utilizes SFML. To compile this code one will need to have SFML downloaded on their machine. The GameLogic class handles all of the logic aspects of the connect four game. Lastly, the main.cpp file instantiates these objects and runs the game.
